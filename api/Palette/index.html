Generates a TailwindCSS-like palette from a given base color. The generator is based on Smart Swatch by Ivan Dalmet, so results will not be identical to TailwindCSS. 

local BASE_COLOR = Color3.fromHex("#00a2ff")
local palette = Tailwind(BASE_COLOR)
